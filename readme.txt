plugins.infosfttech.com presents Sales Trends Analysis Report Lite, Its basic WooCommerce Sales Trends Analysis Report Plugin to help you more insights on how your sales taking place.

== Description ==
WooCommerce Sales Trend Analysis Reports gives you more insights on how your sales are taking place like what combination/group of products are getting sold, Order Quantity Analysis, Quantity wise Top sales, Value wise Top sales, 0 Sales products.

**Features in Free Version**:
* Dashboard 
* Simple product order quantity analysis
* Variation product order quantity analysis
* Customer Who has not Purchased within particular date range


**Features in  Pro Version**:
*New Customer/Repeat Customer Analysis 
* Top n Customer Report who Orders Frequently
* Product Combination/ Group sales analysis
* Customer in Price Point i.e. Search on min and max price range report
* Top n Customer Report who Orders Frequently

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins > Add New > Search` and search for 'Sales Trends Analysis for WooCommerce').
* Alternatively, you can [download](http://downloads.wordpress.org/plugin/ic-sales-trends-analysis-report-lite.zip) the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* Access the settings page from WordPress backend under Sales Trends Analysis.
